🛠️ Instrucciones paso a paso

  1. Descarga el código: Clona este repositorio en tu computadora o descarga el archivo ZIP.
  2. Prepara tu entorno: Abre la carpeta en tu editor de código, por ejemplo VS Code.
  3. Analiza el HTML: Abre el archivo tarea2.html.
  4. Crea tu hoja de estilos: Crea un archivo nuevo llamado tarea2.css.
  5. Conecta los archivos: Vincula tu CSS dentro de la etiqueta <head> usando <link>.
  6. ¡A programar! Escribe tu CSS basándote en la imagen de referencia.

✅ ¿Qué se evaluará?

  • Similitud visual: Colores, márgenes, padding y alineaciones parecidos al diseño esperado.
  • Uso correcto de selectores: Buen manejo de clases y selectores de etiqueta.
  • Código limpio: CSS ordenado, legible e indentado correctamente.
  • Uso de Flexbox: Aplicación adecuada de herramientas modernas de maquetación.
